Implementation Full Javascript du Monopoly. 
Aucune librairie Javascript et packaging avec Webpack.

h2. Démo

Demo du monopoly  : "Monopoly":http://jotitan.github.io/monopoly/demo/

h2. Fonctionnalités

* Possibilité de jouer à 6 sur le même ordinateur
* IA : plusieurs robots peuvent jouer. Différents niveaux sont implémentés (gestion de l'argent, choix des terrains et agressivité).
* Plusieurs variantes : case départ x 2, parc gratuit, enchere immédiate
* Possibilité d'échanger les terrains avec les autres joueurs
* Utilisation du dé rapide (règle récente)
* Plusieurs plateaux sont disponibles : classique, en cercle
* Possibilité de customiser les plateaux (images, nom des joueurs, nom des rues, cartes, couleurs)
* Sauvegarde la partie (en cliquant sur "MONOPOLY")
* Gestion du réseau (serveur en Go)

h2. Build

Il est possible de tester l'application localement en ouvrant monopoly-template.html
Vous pouvez également l'application pour réduire sa taille en lançant :
```shell
npm run pack
```
